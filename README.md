## 部署
运行环境：`Node.js >= v8.0.0`  
安装：
```
git clone https://github.com/zgq354/weibo-rss.git
cd weibo-rss
npm install
```
启动：  
```
npm install pm2 -g
pm2 start bin/www
```

更多配置参考 [项目Wiki](https://github.com/zgq354/weibo-rss/wiki)
